# Ainul Islam - HTML Portfolio

A standalone HTML portfolio website with all the features of the Next.js version, but in a single file that can be hosted anywhere.

## 🚀 Quick Start

### 1. Download the File
Get `ainul-islam-portfolio.html` from the archive.

### 2. Open in Browser
Simply double-click the HTML file or open it in your favorite web browser.

### 3. That's It!
No installation, no dependencies, no build process required.

## ✨ Features

### 🎯 All Original Features Included:
- **Professional Design**: Modern, responsive layout
- **All Sections**: Home, About, Experience, Education, Skills, Certifications, Contact
- **Working Contact Form**: Integrated with Formspree
- **Interactive Elements**: Smooth scrolling, animations, hover effects
- **Mobile Responsive**: Works perfectly on all devices
- **Live Links**: CV download, website, LinkedIn, profile photo

### 🔧 Technical Features:
- **Single File**: Everything in one HTML file
